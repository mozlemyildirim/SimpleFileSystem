Description:

Simple File System is a program which showing us how running file systems on operating systems. 
Information
This tool is for educational purpose only, usage of our program is like simulating Simple File System.

How To Run?

For Windows:

1-)Run mkdir.exe

2-)Run sfs.exe

3-)Run sim_bash.exe


For Linux:

First, compile on the terminal and then run.

1-)gcc mkdir.exe

2-)gcc sfs.exe

3-)gcc sim_bash.exe

4-) ./a.out
